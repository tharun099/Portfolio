## Overview

The **Tweet Generator Chrome Extension** allows users to select text on any webpage and instantly generate well-crafted tweets while preserving the original context.

Users can:
- Customize the **tone** of the tweet  
- Control the **tweet length**  
- Regenerate tweets for variations  
- Copy tweets with a single click  
- Track real-time character counts  

All of this happens inside a clean and intuitive Chrome extension popupâ€”no tab switching required.

## File Structure & Purpose

### `manifest.json`
- Chrome extension configuration file  
- Defines permissions, content scripts, and metadata  
- Registers the popup interface  

---

### `src/components/TweetGenerator.tsx`
- Main UI component for tweet generation  
- Manages:
  - Selected text
  - Tweet options (tone and length)
  - Generated output  
- Handles API calls and user interactions  
- Renders controls and tweet preview  

---

### `src/lib/gemini.ts`
- Integrates **Google Gemini AI**  
- Formats prompts for tweet generation  
- Processes AI responses  
- Handles API errors gracefully  

---

### `src/content.tsx`
- Content script injected into web pages  
- Detects text selection events  
- Communicates selected text to the extension  
- Uses `chrome.storage` for persistence  

---

### `src/types.ts`
- Centralized TypeScript definitions  
- Defines interfaces for:
  - Tweet options
  - API responses  
- Stores constants for tone and length options  

---

### `src/App.tsx`
- Root component of the extension  
- Provides layout and structure  
- Handles component organization  

---

### `vite.config.ts`
- Build configuration using Vite  
- Optimizes assets for production  
- Supports fast development and extension packaging
